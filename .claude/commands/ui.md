# 🎨 Command: /ui (UI Engineering)

## 🎯 Purpose
Активация режима проектирования высококачественных интерфейсов на базе библиотек **uipro** и **21st.dev Magic**.

## 🛠 Workflow
1. **Trigger**: Пользователь вводит `/ui [описание компонента или экрана]`.
2. **Analysis**: Агент анализирует запрос на соответствие `SPECIFICATION.md` и стилю "Analytic Noir".
3. **Execution**:
   - Поиск шаблонов через `uipro-cli`.
   - Поиск атомарных элементов через MCP-сервер `magic-ui` (21st.dev).
4. **Integration**: Предложение кода компонента с автоматической настройкой Tailwind-классов и анимаций Framer Motion.

## 📜 Constraints
- **Fidelity**: Любой сгенерированный через `/ui` код должен строго следовать сетке 8pt и цветовым токенам из спецификации.
- **Verification**: После вставки компонента агент обязан предложить запуск Playwright для визуальной проверки.

## ⌨️ Example Usage
- `/ui создать дашборд тренировок в стиле Analytic Noir`.
- `/ui найти анимированную карточку для списка гитарных аккордов`.
