# Command: /think

Когда пользователь вводит эту команду:
1. **Протокол**: Активируй MCP сервер `sequential-thinking`.
2. **Процесс**: Разбей сложную проблему на логические шаги, проверяя каждое предположение на ошибки.
3. **Результат**: Выдай обоснованное решение или план действий для Planner или Executor.
